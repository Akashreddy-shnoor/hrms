# SHNOOR HRMS

Full-stack HR Management System built with React + Node.js + PostgreSQL.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Auth**: JWT

## Project Structure
shnoor-hrms/
├── hrms-frontend/   → React frontend
└── hrms-backend/    → Node.js backend
